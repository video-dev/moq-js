MoQ Player - Samples

  • Web Component - Basic Example

    Example of how to implement the moq-player web component.

  • Web Component - Advanced (Modules) Example

    Example of how to use moq-player as a module.

  • Media Chrome - Example

    Shows how to use the MoQ player with Media Chrome controls and themes.

  • Simple Player - Example

    Example of how to directly use the player without the built-in visual controls. You need to implement them on your own.

  • Event Handling - Example

    Demonstrates how to listen for and handle events emitted by the MoQ Player to create custom interactions.

  • Frame Burst Test

    Diagnostic test to measure and demonstrate the frame burst problem at playback start.
